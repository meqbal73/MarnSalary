# CALREN — Supabase Salary Dashboard

واجهة جديدة بالكامل لحاسبة الراتب بالساعات وإدارة الشفتات، مع الإبقاء على اتصال Supabase ومنطق الحفظ والمزامنة كما هو.

## الواجهة الجديدة
- تصميم قريب من النماذج المرفقة مع بطاقات شفافة وخلفية متدرجة.
- لونان قابلان للتبديل: **Emerald Green** و **Purple**.
- وضعان: **Dark** و **Light**.
- تصميم متجاوب للجوال والتابلت واللابتوب والشاشات الكبيرة.
- تقويم شهري يعرض أيام الشفتات والساعات.
- إضافة سريعة للشفت من الصفحة الرئيسية، والقيم الافتراضية 5 ساعات و25 ريالًا للساعة.
- ملخص للشهر وجدول للشفتات وتسجيل المبلغ المستلم.
- قائمة حساب للنسخ الاحتياطي JSON والاستيراد وإعادة التعيين وتسجيل الخروج.
- دعم العربية والإنجليزية وRTL/LTR.

## الباك إند
لم يتم تغيير إعدادات Supabase أو طريقة تسجيل الدخول أو بنية البيانات أو عمليات القراءة والحفظ. ملف `config.js` بقي كما هو.

## الملفات
- `index.html`: الواجهة ومنطق التطبيق.
- `styles.css`: التصميم المتجاوب والثيمات.
- `config.js`: بيانات اتصال Supabase الحالية.
