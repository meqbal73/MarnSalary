# CALREN

حاسبة راتب مرنة تعتمد على ساعات العمل اليومية وسعر الساعة. تعمل محليًا دون خادم وتحفظ البيانات في المتصفح.

## التشغيل
افتح `index.html` مباشرة، أو ارفع الملفات إلى GitHub Pages.

## المميزات
- إدخال سريع للشفتات.
- تقويم ميلادي مع ساعات كل يوم.
- منع اليوم السابع المتتالي تلقائيًا.
- مقارنة المستحق بالمبلغ المستلم.
- إغلاق الشهر بعد استلام الراتب.
- إضافة شهر عمل كامل أو حذفه.
- تصدير واستيراد JSON.
- عربي/إنجليزي ووضع ليلي/فاتح.
