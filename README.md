# Marn Local Dashboard

واجهة مستقلة لإدارة دخل مرن فقط.

- لا تستخدم Firebase أو أي قاعدة بيانات خارجية.
- البيانات محفوظة في `localStorage` داخل نفس المتصفح والجهاز.
- عند فتح النسخة لأول مرة، يمكنها نسخ بيانات الشهور من مفتاح المشروع القديم `my-finance-data-v2` إن كان موجودًا، ثم تحفظ مرن بشكل مستقل في `marn-dashboard-local-v1`.
- افتح `index.html` مباشرة أو ارفع الملفات إلى GitHub Pages.

> ملاحظة: مسح بيانات المتصفح أو التخزين المحلي يؤدي إلى حذف البيانات المحلية.
