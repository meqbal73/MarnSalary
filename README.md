# Marn Local Dashboard

واجهة مستقلة لإدارة دخل مرن فقط.

## المزايا
- تبدأ بدون أي بيانات افتراضية.
- حفظ البيانات محليًا داخل نفس المتصفح باستخدام LocalStorage.
- إدخال شفت واحد في كل مرة.
- إدخال شهر كامل دفعة واحدة من نموذج واحد متعدد الصفوف.
- حساب المستحقات والساعات تلقائيًا.
- تسجيل مبلغ الاستلام وحالة تم الاستلام.
- إضافة وتعديل وحذف الشهور والشفتات.
- وضع فاتح وداكن.

> البيانات لا تنتقل تلقائيًا إلى جهاز أو متصفح آخر، وقد تضيع عند حذف بيانات الموقع من المتصفح.
